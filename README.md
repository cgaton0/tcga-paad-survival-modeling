# TCGA-PAAD Survival Modeling

Survival prediction modeling in pancreatic cancer using clinical and genomic TCGA data.

## Project Overview

This project reanalyzes TCGA-PAAD clinical and genomic data (185 patients) to identify prognostic factors and develop individualized survival prediction models.

The study integrates traditional survival analysis methods with neural survival modeling approaches.

## Objectives

- Identify clinical and genomic prognostic factors  
- Perform survival analysis using Kaplan–Meier and Cox regression  
- Develop and compare statistical and neural survival models  
- Evaluate predictive performance using Concordance Index (C-index)  
- Assess model interpretability with SHAP  

## Methodology

1. Data extraction and preprocessing of multi-source clinical and mutation datasets  
2. Exploratory Data Analysis (EDA)  
3. Survival analysis (Kaplan–Meier, Cox regression)  
4. Neural survival modeling using a Cox-based MLP  
5. Model evaluation and interpretability  

## Results

- Competitive survival prediction performance  
- C-index ≈ 0.6–0.7  
- Identification of key genomic mutations (KRAS, TP53, CDKN2A, SMAD4)  
- Interpretable risk contributions via SHAP  

## Tech Stack

- Python  
- Pandas / NumPy  
- Scikit-learn  
- Lifelines  
- TensorFlow / Keras  
- SHAP
